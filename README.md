# 🎥 Learning Management System (LMS)

A modern web-based Learning Platform that allows users to watch video lectures and track their learning progress across different subjects like HTML, CSS, and JavaScript.

This project simulates a real LMS system like online learning platforms.

---

## 🚀 Features

✨ Watch video lectures using YouTube iframe  
✨ Search videos instantly  
✨ Filter videos by subject (HTML / CSS / JS)  
✨ Track learning progress automatically  
✨ Save watched videos using LocalStorage  
✨ Dynamic progress dashboard per subject  
✨ Fully responsive UI for all devices  
✨ Clean modern design (Glassmorphism style)

---

## 🛠️ Built With

- HTML5
- CSS3 (Modern UI + Animations)
- JavaScript (DOM + LocalStorage)

---

## 📊 How Progress System Works

- Each video has a unique ID
- When user clicks **Watch**

  - Video opens in modal (iframe)
  - Video is saved in LocalStorage as "watched"

- Dashboard calculates:
  - Total videos per subject
  - Watched videos
  - Completion percentage (%)

👉 Example:

- HTML → 2/3 videos → 66%
- CSS → 1/3 videos → 33%
- JS → 3/3 videos → 100%
